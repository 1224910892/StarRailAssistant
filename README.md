****使用说明****

**1：分辨率请调为1920*1080 （窗口化）**   

**2：脚本运行时请勿移动鼠标和键盘，避免寻路点位出现偏离** 

**3：强烈推荐使用远程角色开怪**

**4：战斗为游戏自带的自动战斗，确保你的队伍有足够实力平推小怪**

**5：如果脚本无反应，请使用管理员权限运行**
